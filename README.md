\# Enterprise Campus Network Design



A secure multi-campus enterprise network simulation built using Cisco Packet Tracer.



\## Features

\- VLAN Segmentation

\- Inter-VLAN Routing

\- OSPF Dynamic Routing

\- DHCP 

\- IPv4 \& IPv6 Dual Stack

\- DMZ Implementation

\- Static NAT

\- ACL Security Filtering



\## Technologies

\- Cisco Packet Tracer

\- Cisco IOS

\- OSPF

\- NAT

\- ACL

\- Packet Tracer topology (.pkt)

\- Validation screenshots
