# Secure Enterprise Campus Network Design

A secure multi-campus enterprise network simulation built using Cisco Packet Tracer. This project implements enterprise-level network infrastructure concepts including VLAN segmentation, Inter-VLAN Routing, OSPF Dynamic Routing, DHCP & DHCP Relay, IPv4 & IPv6 Dual Stack, Multi-Campus Connectivity, DMZ implementation, Static NAT, ACL Security Filtering, and Redundancy Link to create a scalable, secure, and efficient campus network architecture.

# Network Validation
The network infrastructure has been successfully tested using several validation scenarios including Inter-VLAN Communication, Inter-Building Connectivity, Multi-Campus Routing, DHCP Automatic Address Assignment, OSPF Neighbor Adjacency, DMZ Accessibility, Static NAT Translation, and ACL Security Filtering.

---

# Security Implementation
This project implements several security mechanisms to protect internal campus infrastructure including DMZ network separation, Static NAT for public server access, ACL filtering on the edge router, internal VLAN protection from external access, and VLAN-based network segmentation.
